LLM Telegram Support Bot

Telegram-агент для техподдержки с контекстом диалога и оптимизацией токенов.

## Технологии
- Python 3.11+
- aiogram 3.x
- FastAPI
- PostgreSQL
- OpenAI API

## Локальный запуск
1. `cp .env.example .env` и добавь токены
2. `pip install -r requirements.txt`
3. `python src/bot.py`
